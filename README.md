Commands for Environment Setup:-

1.First Add "uv init"- This will Create the skeleton for our process also creates a Main.py file.
2.Then add "uv add langchain"- Will add Langchain.
3.Add "uv add python-dotenv black isort"- Helps load the Environment Variables, also adds "Black" which formats your code and "Isort" will sort your imports into groups.
4.Also which doing all this dont forget to checkout into a new branch from main.
